# 🌌 Noosphere Protocol

A mycelial protocol for the planetary mind. Built with Next.js 13, TailwindCSS, and TypeScript.

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site.

## 📦 Deployment

Deploy instantly with [Vercel](https://vercel.com).  
Just push this repo to GitHub and import it into Vercel.

## ✨ Features

- Hero section with intro
- Ritual mechanics explanation
- Tokenomics with distribution
- Sacrifice and harvest concept
- Call-to-action to enter NooSpace
